- Проект представляет карьерного бота по карьерному треку и разработан с нуля, 
имеет следующие кнопки:
# Кнопка Анкета по профориентации 
- Представляет собой набор связанных методов,которые делают опрос пользователя. После сбора данных выдается рекомендация по совершенстованию навыков в карьерном треке.

# Кнопка Рекомендации по карьерному треку
- Представляет набор компонентов-кнопок (Собеседование, Резюме, Литература для карьерного роста, Схема карьерного роста) и кнопка Назад.

# Собеседование
- Задает промт для ассистента по подготовке к собеседованию

# Резюме
- Задает промт для ассистента по подготовке резюме. Файл импорта [promt_resume](promt_resume.py)

# Литература для карьерного роста
- Задает промт для ассистента по дополнительной литературе. Файл импорта [prompt_books](prompt_books.py)

# Схема карьерного роста
- Задает промт для ассистента по генерации схемы роадмап специальности.

# Вернуться
- Возвращает на главную страницу.
